Projeto 4: Rede Neural

Este projeto implementa uma rede neural simples em Python. Utiliza bibliotecas como scikit-learn, matplotlib e numpy, além de streamlit para visualização interativa dos dados.

Requisitos para execução:

  • Python 3.10+
  • scikit-learn
  • Streamlit
  • Matplotlib
  • Poetry (para gerenciamento de dependências)

Para iniciar a execução do projeto, utilize o comando:

streamlit run app.py
Ver repositório no GitHub

Projeto 5: IA - Pizza

Projeto de aprendizado de máquina que utiliza diversas bibliotecas populares em Python, como scikit-learn, pandas, matplotlib e streamlit, para realizar análises de dados e modelagem preditiva.

Para rodar o projeto, clone o repositório e instale as dependências com o Poetry:

git clone https://github.com/jadelcruz1/IA---PIZZA.git

Em seguida, execute:

poetry run streamlit run app.py

Projeto 6: SpringScreemMatch

Este é um projeto de backend para gerenciamento de séries, criado com o Spring Framework. O sistema permite o cadastro de séries, atores, temporadas e episódios, proporcionando uma interface para o gerenciamento de dados de uma plataforma de streaming.

Principais tecnologias utilizadas:

  • Spring Boot
  • JPA (Java Persistence API)
  • MySQL
  • Spring Security

Este projeto implementa uma API RESTful para interagir com dados de séries, utilizando as melhores práticas de desenvolvimento para APIs com segurança e persistência de dados.
