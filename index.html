Quem sou eu?

Sou um profissional apaixonado por tecnologia, design e inovação. Ao longo da minha carreira, participei de diversos projetos desafiadores que me ajudaram a crescer e aprender constantemente. Além disso, sou curioso e estou sempre em busca de novos conhecimentos.

Principais Projetos

Aqui estão alguns dos projetos que desenvolvi recentemente:

  • Projeto 1: Todo List - Este é um projeto de lista de tarefas simples, desenvolvido em Java. A aplicação permite adicionar, listar, editar e excluir tarefas, armazenando os dados localmente.
  • Projeto 2: E-commerce API - API criada com Spring Boot Web. Utiliza MySQL como banco de dados e Maven para gerenciamento de dependências. Implementação do Swagger para documentação da API e uso de Spring Security para segurança.
  • Projeto 3: Flutter App - Projeto inicial desenvolvido com Flutter. A aplicação serve como ponto de partida para quem deseja aprender a desenvolver aplicativos móveis com Flutter.
